
Workshop 14 : Create your deployment dockers

1. Create new git repositories and checkout to projects dir project/laravel-quickstart-dockers

2. Create branch internal and checkout

3. Copy files from 14-deployment-dockers/internal to project/laravel-quickstart-dockers/dockers

4. Commit and push internal branch

5. Checkout master branch

6. Then create production branch and checkout

7. Copy files from 14-deployment-dockers/production to project/laravel-quickstart-dockers/dockers

8. Commit and push production branch

9. Now you have dockers container configuration for internal and production ready to deploy
